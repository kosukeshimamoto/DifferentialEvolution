# Reports

This folder keeps the latest benchmark results and visualizations so they can be
reviewed without re-running scripts. Outputs are generated by the scripts in
`/scripts` and are kept intentionally for release.

## 2D visualizations

- `reports/2d/2d_gallery.html`
- `reports/2d/*.gif`

Reproduce:

```bash
julia scripts/plot_2d_exploration.jl
```

## Benchmarks

### Algorithm comparison

- `reports/benchmarks/algorithm_compare/benchmark_summary.csv`
- `reports/benchmarks/algorithm_compare/benchmark_summary.md`
- `reports/benchmarks/algorithm_compare/benchmark_overview.html`
- `reports/benchmarks/algorithm_compare/plots/*.png`
- `reports/benchmarks/hybrid_de_local_benchmark.md` (detailed hybrid DE + local-refinement report)

Reproduce:

```bash
julia scripts/benchmark_compare.jl
Rscript scripts/plot_benchmark_bars.R --csv reports/benchmarks/algorithm_compare/benchmark_summary.csv --outdir reports/benchmarks/algorithm_compare/plots
```

Key env vars:
- `DE_BENCH_MAXEVALS`, `DE_BENCH_RUNS`, `DE_BENCH_SEED`
- `DE_BENCH_F`, `DE_BENCH_CR`, `DE_BENCH_PMAX`
